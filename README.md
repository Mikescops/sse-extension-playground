# Server Sent Events in Extension - Playground

This repository aims to show the use of server sent events within an extension.

---

### How to run server

```
cd server
npm ci
node server.js
```

### How to build the extension

```
npm ci
npm run build
```

### How to ship build

```
npm run zip
```
